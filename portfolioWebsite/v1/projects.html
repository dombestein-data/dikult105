learn.dombesteindata.net

learn.dombesteindata.net is a lightweight educational platform for teaching web development. Originally built to support a introductory crash course in Web Development as part of the University of Bergen's DIKULT105 course, it focuses on hands-on, framework-free learning - perfect for students exploring code and interactivity for the first time.

Stack Highlights

  • Astro (Static-first architecture)
  • React components for interativity
  • TailwindCSS styles
  • Hosted on Cloudflare Workers

Focaline (In Development)

Focaline is a cross-platform productivity app that brings together calendars, notes, emails and real-time collaboration into one workspace. Designed to reduce context-switching and improve focus, Focaline offers a clean interface, fast syncing and AI-assisted organization for busy individuals and teams.

Stack Highlights:

  • Frontend: Astro + React + Tailwind (with shadcn/ui components)
  • Backend: Hono (on Cloudflare Workers) + Supabase Auth
  • Database: PostgreSQL via Supabase
  • General Features: Real-time syncing, Fullcalendar integration, AI-powered notes

Markwright

A Node.js CLI tool for converting Markdown files into styled A4 PDFs with syntax highlighting and GitHub Dark theme support. Uses puppeteer for headless rendering.

Stack Highlights:

  • Node.js + marked, marked-highlight & highlight.js
  • Puppeteer for headless rendering and PDF support
  • GitHub Dark theme for syntax highlighting
  • CLI-based input/output file handling
  • Output: Styled A4 PDF with accurate markdown formatting

JCompressor

A Node.js CLI tool for compressing files, folders and images into zip archives with customizable strength and a focus on simplicity. Still in Alpha, but fully working and published on GitHub.

Stack Highlights:

  • Node.js with yazl, fs and async recursion
  • File and directory compression using ZIP format
  • No external dependencies except for Node core and yazl

Velox (In Development)

Velox is a modern, high-performance ERP system designed to replace legacy tools like Microsoft Dynamics AX. Built with a focus on speed, simplicity and seamless workflow, Velox helps retail and logistics companies manage inventory, orders and vendor relationships efficiently. The system supports gradual data migration and continuous sync from older platforms to ease transitions.

Stack Highlights:

  • Frontend: Astro + React + Tailwind (with shadncn/ui components)
  • Backend: Hono (on Cloudflare Workers)
  • Database: PostgreSQL via Neon with Drizzle ORM
  • Other: UploadThing for File uploads, Axion for Logging
